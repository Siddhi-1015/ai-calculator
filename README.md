### 🤖 Futuristic AI Calculator
> HTML · CSS · JavaScript · Voice Recognition

A fully responsive AI-inspired calculator with voice input,
keyboard support, and smooth animations — built with
pure HTML, CSS and JavaScript. No frameworks used.

**Features:**
- 🎤 Voice input using SpeechRecognition API
- ⌨️ Full keyboard support
- 📱 Fully responsive on all devices
- 🎨 Animated gradient background
